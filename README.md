# Kayali Commercial Intelligence Analysis 2025
Independent commercial intelligence analysis of Kayali Fragrances 2025

## Overview
An independent commercial intelligence analysis of Kayali Fragrances 
built as a portfolio project to demonstrate data analyst capabilities from simulated data.

## What This Project Covers
- Multi-market pricing analysis across US, UK and UAE
- Competitor benchmarking against 6 luxury fragrance brands
- Promotional effectiveness and ROI analysis across 6 campaigns
- Market expansion opportunity scoring for 6 markets
- Price elasticity scenario modelling

## Tools Used
- Python (Pandas, NumPy, Matplotlib, Seaborn)
- Microsoft Power BI
- Microsoft Excel

## Files
- /notebooks — Python analysis notebooks
- /data — Source data
- /outputs — Analysis output CSV files
- /dashboard — Power BI report (.pbix)
- /executive_summary — One page PDF summary

## Key Findings
- Kayali generated $229.4M in estimated 2025 revenue
- Vanilla 28 contributes 32.4% of total SKU revenue
- Only Black Friday (163.52% ROI) and Valentine's Day (132.25% ROI) 
  generated positive promotional returns
- France identified as highest priority expansion market

## Data Note
All sales data is simulated based on publicly available market 
information. Competitor pricing collected from live retail websites 
across US, UK and UAE markets in April 2026.
